# WEB-INTRO-LAB Internet và Web

Phân biệt Internet và Web

Internet là hệ thống mạng toàn cầu kết nối nhiều thiết bị với nhau để trao đổi dữ liệu và thông tin.

World Wide Web (WWW) là một dịch vụ hoạt động trên Internet, cho phép người dùng truy cập các trang web bằng trình duyệt.

Một số khái niệm liên quan

  • Website: Tập hợp nhiều trang web
  • Trang web: Một trang nội dung cụ thể
  • Trình duyệt: Phần mềm dùng để truy cập web
